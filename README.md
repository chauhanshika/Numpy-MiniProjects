# NumPy Mini Projects

This repository contains a collection of **10 hands-on mini projects built using NumPy** to strengthen my understanding of numerical computing, array manipulation, and data analysis with Python.

The goal of this repository is to practice **vectorized operations, matrix computations, statistical analysis, and algorithmic thinking** using NumPy before moving to full data science workflows with Pandas and machine learning tools.

---

## Projects Overview

| Project    | Topic                        | Key Concepts                   |
| ---------- | ---------------------------- | ------------------------------ |
| Project 1  | Student Marks Analyzer       | axis operations, aggregation   |
| Project 2  | Weather Data Analyzer        | filtering, statistical metrics |
| Project 3  | Sales Data Analyzer          | multidimensional arrays        |
| Project 4  | Image Processor              | matrix manipulation            |
| Project 5  | Stock Data Analyzer          | time-series statistics         |
| Project 6  | Recommendation System        | cosine similarity              |
| Project 7  | Linear Regression            | numerical modeling             |
| Project 8  | Principal Component Analysis | covariance, eigenvalues        |
| Project 9  | Mini Neural Network          | matrix multiplication          |
| Project 10 | Large Dataset Simulator      | performance optimization       |
